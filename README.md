## Como usar

- Clone o repositório

```bash
git clone https://github.com/luizfelmach/boca-dump
cd boca-dump
```

- Instale as dependências

```bash
python -m venv .venv #(opcional)
source .venv/bin/activate #(opcional)
pip install -r requirements.txt
```

- Rode o programa

```bash
python3 boca-dump.py output_path
```
